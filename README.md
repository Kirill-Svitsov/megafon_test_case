## Описание

Проект представляет собой API для работы с геоданными на основе библиотеки H3. Основные функции:

- Получение данных по H3 гексагонам
- Агрегация данных по медиане
- Генерация KML файлов с гексагонами
- Работа с географическими полигонами

## Стек технологий

- Python 3.12
- FastAPI
- H3 (библиотека для работы с гексагональными сетками)
- Uvicorn (ASGI сервер)
- Pydantic (валидация данных)
- Black и Flake8 (линтеры)
- Pytest (тестирование)

## Установка и запуск

1. Клонируйте репозиторий
2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   venv\Scripts\activate     # Для Windows
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```
   Приложение будет доступно по адресу http://localhost:8888

## API Endpoints

### 1. Получение данных по H3 гексагону

```
GET /hex?hex={h3_index}
```

### 2. Агрегация данных по медиане

```
GET /avg?resolution={resolution}&limit={limit}&offset={offset}
```

### 3. Получение гексагонов внутри полигона

```
GET /bbox?border={border_string}&limit={limit}&offset={offset}
```

### 4. Генерация KML файла с гексагонами

```
GET /bbox_kml?border={border_string}
```

### 5. Диагностика производительности

```
GET /avg_benchmark?resolution={resolution}
```

## Примеры запросов/ответов

### Пример 1: Получение данных по гексагону

**Запрос:**

```
GET /hex?hex=8c11aa6483601ff
```

**Ответ:**

```json
[
  [
    "8c11aa6483601ff",
    -88,
    93
  ]
]
```

### Пример 2: Агрегация данных по медиане

**Запрос:**

```
GET /avg?resolution=5&limit=10&offset=0
```

**Ответ:**

```json
  "offset": 0,
  "has_next": true,

      -88,
      1
    ],
    [
      "8511aa6483601fe",
      -87,
      2
    ]
  ]
}
```

### Пример 3: Получение гексагонов внутри полигона

```json
  "offset": 0,
  "has_next": false,

    [
      "8c11aa6483601ff",

      93
```

### Пример 4: Генерация KML файла с гексагонами

**Запрос:**

```
GET /bbox_kml?border={border_string}
```

**Ответ:**

Аналогичен /bbox, но возвращает KML файл для скачивания.

### Пример 5: Диагностика производительности

**Запрос:**

```
GET /avg_benchmark?resolution={resolution}
```

**Ответ:**

Возвращает метрики производительности без передачи данных (для отладки).

## Оптимизация

### Пагинация

* Эндпоинты /avg и /bbox поддерживают пагинацию через параметры limit и offset, что позволяет получать данные порциями и
  избегать больших ответов.

### Кэширование

* Датасет: генерируется один раз при первом запросе и кэшируется через @lru_cache
* Результаты /bbox: кэшируются для каждого уникального полигона. Первый запрос занимает ~3-5 секунд, последующие — менее
  1 мс

## Тестирование и линтеры

### Структура тестов

- test_h3_data.py — тестирование H3 функций (парсинг, геометрия, медиана)
- test_main.py — тестирование всех эндпоинтов (включая пагинацию, CORS, ошибки)
- conftest.py — фикстуры для клиента и тестовых полигонов
- Запуск тестов

```
pytest tests/ -v
```
